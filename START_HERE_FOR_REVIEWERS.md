# Start here for reviewers

This repository presents a structural-functional parser model of Voynichese.

The claim is not a normal word-for-word translation.
The claim is that Voynich tokens function as compressed operator chains.

## Minimum things to inspect

1. README.md — claim hierarchy and witness protocol
2. RJN Voynich Parser.txt — full grammar paper
3. Translation how to use parser — practical parsing guide
4. parse tree.txt — worked structural examples
5. update scripts april 5.txt — computational development log
6. voynich_corpus.txt — working corpus layer

## Core falsifiable claims

- q behaves as initiation / active process marker
- y behaves as terminal / closure marker
- d compounds behave as transformation/closure structures
- o forms process-field families such as ok/ot/ol/or/op
- tokens parse better as operator chains than as lexical words
- sequence-level parsing explains structures that word-by-word decoding does not
